# 从源文件到可执行文件

## 过程

gcc -E main.c -o main.i 预处理 </br>
gcc -S main.i -o main.s 编译 </br>
gcc -c main.s -o main.o 汇编（注意-c是小写） </br>
gcc main.o    -0 hello  链接 </br>

## 注释

```c
//单行注释
/*
多行注释
*/
#if 0
调剂编译
#endif
```

## 生成文件

Linux      main.c ->  a.out </br>
Windows    main.c ->  a.exe </br>
