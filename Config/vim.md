# vim config

在当前用户home目录中</br>

设置本用户，输入命令

```
vim ~/.vimrc
```

设置全局用户，输入命令

```
vim /etc/[vim/]vimrc
```

 打开文件后，输入内容

```
set number                                                                      syntax on                                                                        set cindent                                                                      set autoindent                                                                  set shiftwidth=4                                                               set tabstop=4                                                                set softtabstop=4
```

  保存即可
