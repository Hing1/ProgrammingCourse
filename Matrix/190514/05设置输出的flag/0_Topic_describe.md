# 设置输出的flag

## Description

将输入的数按照系统默认格式，定点格式显示浮点数。

## Input

第一行N代表接下来会有N个cases。</br>
接下来N行中，每行包含一个浮点数。</br>

## Output

对每个浮点数按照 系统默认格式，定点格式分行显示。

## Sample_Input

```cpp
4
2123.324423
2123.324423
2123.324423
123.0000
```

## Sample_Ouput

```cpp
2123.32
2123.324423
2123.32
2123.324423
2123.32
2123.324423
123
123.000000
```

## Hint

重设格式
